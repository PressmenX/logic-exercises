## 📚 Daftar Penguasaan
- Logic & Conditionals: Penggunaan if-else bertingkat, Guard Clauses, dan operator pembanding.
- Arithmetic & Modulo: Penggunaan Modulus (%) untuk menentukan kelipatan, sisa bagi, dan validasi angka.
- Looping: Perulangan dasar for untuk iterasi array dan pembentukan pola.
- Nested Looping: Logika perulangan di dalam perulangan untuk membentuk pola visual.
- String Manipulation: Penggabungan karakter, pembalikan teks, pemotongan (slice), pencocokan pola (replace, regex), dan pemrosesan urutan teks.
- Array Manipulation: Pembalikan urutan, penambahan elemen, pencarian nilai, dan perataan array bertingkat.
- Object & Grouping: Penyimpanan data dengan key-value dan pengelompokan data berdasarkan kategori.
- Functional Programming: Penggunaan reduce untuk agregasi, transformasi, dan grouping data sekaligus dalam satu iterasi.
- Accumulator Pattern: Teknik membangun hasil secara bertahap menggunakan variabel penampung di setiap iterasi.
- Temp Variable Pattern: Teknik menukar nilai dua variabel menggunakan variabel sementara sebagai penampung.
